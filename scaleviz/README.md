# ScaleViz — AI Scale-Up/Out 交互式教学博客

> "Don't just explain — let them play with it."

## 开发

```bash
npm run dev    # http://localhost:3000
npm run build  # 生产构建
```

## 技术栈

- Next.js 15 (App Router, RSC, Turbopack)
- TypeScript 5
- Tailwind CSS v4
- next-intl (中/EN 双语)
- MDX (next-mdx-remote)
- Framer Motion 12
- Shiki (代码高亮)
- KaTeX (数学公式)
